# ENVS-193DS_homework-03

## General Information 

This repository is for ENVS 193DS Homework 03. Homework 03 consists of statistical analysis of soil salinity and California pickleweed biomass using correlation and regression tests, personal data visualization updates, an affective data visualization draft and artist statement, and a statistical critique of a peer-reviewed paper.
This repository belongs to me, Macey Hartmann, and I am the sole contributor.

## Data and File Information

The data folder includes a salinity-pickleweed.csv, provided by An Bui in ENVS 193DS. The folder also includes a Personal_Data_ENVS194DS.csv that I collected on my own. 

The code folder includes an index.qmd and index.html which contain the code for analyzing the picklweed salinity data, and creating visualizations for my personal data.

The images folder includes the images for my personal data affective visualization sketch and draft and an image of Figure 3 from the publication used in my statistical critique.
